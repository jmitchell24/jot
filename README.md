## Build Instructions

```bash
# Clone the repository
git clone https://github.com/jmitchell24/jot.git

# Navigate to the project directory
cd jot

# Make build directory
mkdir build
cd build 

# Generate build files / build project  
cmake .. 
cmake --build . 

```

## Usage

### Basic Usage

```bash
jot
```
